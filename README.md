## 功能介绍（完整见运行截图）
管理员：基本功能包括员工信息的增加、删除、修改和查找。用户可以通过注册和登录功能访问系统，并进行安全登录。界面设计简洁，支持用户高效管理员工数据。

用户：可以通过注册新账号或使用已有账号进行登录。登录后，用户能够查看和编辑个人信息，并通过系统提供的功能对员工信息进行管理和操作。
